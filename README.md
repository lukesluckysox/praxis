# Praxis

**The Laboratory of the Self**

Praxis is the third tool in an introspection ecosystem alongside Liminal and Parallax. Where Liminal interrogates beliefs and Parallax reveals patterns, Praxis transforms insight into lived experimentation.

---

## What it is

Praxis is a personal behavioral laboratory. It treats the user's life as a series of experiments designed to test the truths they think they believe.

Every experiment follows this structure:

```
Hypothesis → Experiment → Observation → Meaning Extraction
```

Over time, completed experiments accumulate into **Working Doctrines** — provisional truths derived from lived experience rather than theory — and surface **Core Tensions** — the gravitational poles shaping how a person navigates their life.

---

## The three areas

- **Active Experiments** — trials in progress, each with a hypothesis, design, constraint, observation record, and meaning extraction
- **Emerging Doctrines** — principles extracted from completed experiments (emerging → established → superseded)
- **Core Tensions** — life axes identified through patterns (e.g., Autonomy ↔ Recognition)

---

## What it is not

- A habit tracker
- A productivity app
- A motivational coach
- A wellness dashboard
- An advice engine

---

## Tech stack

- **Frontend**: React + TypeScript + Tailwind CSS + shadcn/ui
- **Backend**: Express + SQLite (via better-sqlite3 + Drizzle ORM)
- **Fonts**: Zodiak (display) + General Sans (body) via Fontshare
- **Build**: Vite

---

## Running locally

```bash
npm install
npm run dev
```

The app runs on [http://localhost:5000](http://localhost:5000).

| Tool | Role |
|------|------|
| **Liminal** | Interrogates beliefs |
| **Parallax** | Reveals patterns in writing and behavior |
